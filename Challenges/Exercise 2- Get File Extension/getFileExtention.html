Get file extension (JavaScript Challenge)

            

                Write a JavaScript program to get the
                extension of a filename.
            

            

                String Method:
                    .split()- splits a string into an array of substrings.
                            - method returns the new array
                            - method does not change the original
                                Syntax:
                                    string.split(separator, limit)

                                        Parameters:
                                            separator: Optional
                                                A string or regular expression to use for splitting.
                                                If omitted, an array with the original string is returned.
                                            limit: Optional
                                                An interger that limits the number of splits.
                                                Items after the limit are excluded.
                String or Array Method:
                    .lastIndexOf()- returns the last index (position) of specified value.
                                                - returns -1 if the value is not found
                                                - starts a specified index and searches from right to left
                                                - by default the search starts at the last element and ends at the first
