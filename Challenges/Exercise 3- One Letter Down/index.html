Write a JavaScript program to replace every character in a given string with the character following it in the alphabet.

        

        //Note:
            //"str" is the general abbreviation for string in JavaScript
            //"chr" is the general abbreviation for characters in JavaScript
            
        //Create a function
                                //passes in a string(which is the parameter of the function)
        const moveCharsForward = (str) =>  
        str                              //since we want to modify the contents of a string
        .split('')  //every letter       //split returns every character as an array
        .map(char =>                     //map loops through the given array(s) one at a time
            String.fromCharCode           //converts the numbers back to letters
            (char.charCodeAt(0) + 1))     //this gives the character a numerical value and increases it by 1
        .join('');                       //since we have our string split into an arrays we need to join it back together
