JavaScript Exercise No. 4

(Get Current Date)

            

                Write a JavaScript program 
                    to get the current date
                Expected Output: 
                    mm-dd-yyyy, mm/dd/yyyy or dd-mm-yyyy, dd/mm/yyyy 
           
                Methods to consider:
                    - new Date()            //create as the original variable
                    - .getFullYear()        //extract the following data from it
                    - .getMonth()
                    - .getDate()
                        which gives you the number day of the month (1-31)
                Notes:
                    * step 1: gather all the dates
                    * step 2: concatinate the data as a single String
                    * step 3: concatinate with a particular divider (either slashes(/) or dashes(-) )
                        this would actually be an Array Method
                            .join()
