(Alphabetize a given word)

        Objective:
            Write a JavaScript function that returns a 
            passed string with letters in alphabetical order.

            Example string: 'webmaster'
            Expected Output: 'abeemrstw'

        Notes:
        Steps:
            1) convert the string into an array 
                .split()
            2) convert the letters to their Unicode numerical value 
                .charCodeAt()

            3) perform a bubble sort algorithm, to rearrange the numbers in ascending order 
            4) convert them from their numeric value to their letter value 
            5) convert the array back into a string 
                .join() or .toString()
