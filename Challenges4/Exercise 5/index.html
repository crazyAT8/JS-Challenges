JavaScript Exercise/Challenge

(Elements Pass the Test)

        Objective:
            Write a JavaScript function that 
                returns true if the provided predicate function returns true 
                    for all elements in a collection, false otherwise.
        Notes:
        Steps:
            - Create an Array 
            - attach the Array to .every() 
            - as an argument, us a function that returns a boolean comparison
                conditional statement comparing the Array 
                    with a particular condition
        Code:
                // My Way
                    const ages = [32, 41, 31, 21, 88, 45];
                    document.getElementById("example1").innerHTML = ages.every(checkAge);

                    function checkAge(age) {
                        return age >= 20;
                    }
                
                    const checkAge2 = age => age == 16;
                    const ages2 = [ 23, 44, 98, 34, 66];
                    document.getElementById("example2").innerHTML = ages2.every(checkAge2);
